Methodology

方法论

所有投研产出遵循统一框架。不是新闻摘要 — 每篇报告必须有分析框架、量化推导、三层结论。

Layer 1 · Conviction

物理事实，不变信念。被证伪 → 全部离场。

→

Layer 2 · 赛道假设

可证伪的预测。有验证指标、拐点、检查日期。

→

Layer 3 · 标的判断

具体操作。标注依赖哪个假设，假设变判断变。

🎯

TAM × 利润率 × 市占率

自上而下量化。赛道有多大，谁能吃多少，利润怎么分。

🏰

AI 时代护城河

四标准：物理稀缺性、数据活性、切换成本深度、标准制定权。

🔄

贝叶斯更新

新证据 → 更新假设概率 → 调整判断。Conviction 没变就不改。

🎰

赔率 > 胜率

不追求高胜率，追求高赔率。用 99% 置信度最大亏损算仓位。

📊

信号灯系统

🟢正常 / 🟡冻结 / 🔴分 3 次减仓。不在压力下决策。

🔍

复盘框架

好机会没参与 → 为什么错过。坏机会参与了 → 为什么误判。
